# FHPoker

FHPoker is aTexas Hold'em with Python server and react FE.
You can play a simple game with it.

## Requirement

MongoDB
Python3^
NPM

## Installation

```bash
pip3 install -r requirements.txt
```

```bash
cd client
npm install
```

## Usage
Start up MongoDB

```bash
python3 app.py
```
Open a new tab
```bash
cd client
npm start
```
